# Market Reports

每日股票市场汇总报告，由 AI 自动生成。

## 目录结构

```
market-reports/
├── daily/              # A股每日收盘汇总
│   ├── 2026-04-15.md
│   ├── 2026-04-16.md
│   └── ...
├── us-stock/
│   └── daily/          # 美股每日收盘汇总
│       ├── 2026-04-15.md
│       ├── 2026-04-16.md
│       └── ...
└── weekly/             # A股每周行情周报
    ├── 2026-W16.md
    └── ...
```

## 报告内容

**A股日报** (`daily/YYYY-MM-DD.md`)
- 主要指数收盘数据（上证、深证、创业板、科创50 等）
- 板块涨跌排行与资金流向
- 热点题材分析
- 技术面与消息面综合研判

**美股日报** (`us-stock/daily/YYYY-MM-DD.md`)
- 三大指数收盘（道琼斯、标普500、纳斯达克）
- 板块轮动与个股亮点
- 宏观消息面
- 技术分析与后市展望

**A股周报** (`weekly/YYYY-Www.md`)
- 本周主要指数表现汇总
- 板块周度涨跌排行
- 本周大事回顾
- 下周展望

## 自动化

报告由 [Hermes Agent](https://github.com/huidge/hermes-skills) 自动生成并同步：

| 报告 | 生成时间 | 同步方式 |
|------|----------|----------|
| A股日报 | 工作日 15:30 | 生成后自动 git push |
| A股周报 | 周五 17:00 | 生成后自动 git push |
| 美股日报 | 每天 06:00 | 定时 06:30/16:30 同步 |

## 数据来源

- A股：东方财富、证券时报数据宝、新浪财经
- 美股：Yahoo Finance、CNBC、Reuters
